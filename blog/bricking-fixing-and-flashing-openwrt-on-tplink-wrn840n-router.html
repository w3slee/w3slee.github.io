Bricking ,Fixing and Flashing OpenWRT on TP-LINK WRN840N Router

Home

This chapter is brought to you by... **"how not to flash openWRT"**. After trying to flash an openWRT build using the web interface of the router, the device got soft bricked and started flashing the led status lights on it. It wasn't able to fully boot up, hence shit wasn't working. After looking up fixes and how to's . I didn't get anything specific to the **TP-LINK WRN840** Series .

This was an exciting short trip of another 'breaking it to learn it'. This was my approach.

The setup

  1. Linux environment
  2. TFTP server
  3. Stock firmware for the router downladed from TP-Link official webpage for the WRN840Nv4
  4. OpenWRT firmware binary image file for the router
  5. An ethernet cable / RJ45 cable
