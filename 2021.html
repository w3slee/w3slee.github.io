    ADB - fun times with the ultimate android tool

    [Home&mdash]

    Everytime I see apps that enable you to do full system backups, screencaptures and screenshots, systems logs, uninstalling system software ( bloatware ) but requires root ; I'm like "adb can do this shit flawlessly". Welcome to the land of the Android Debug Bridge. From automatically swiping on your tinder profiles to recording a gameplay adb can do it. perks of using adb :

    • perform full system backups
    • debug android system issues
    • poke android logs / internals
    • install / uninstall applications
    • pulling application databases ( eg while doing forensics or reversing an application )
      etc...
