Android Automation with ADB

[🏡—🏡base]

adb shell screenrecord

adb shell screenrecord /sdcard/test.mp4 
Records the device's display to a .mp4 file
adb pull /sdcard/test.mp4 
 Download test.mp4 file from Android Device.
adb shell screenrecord [options]

Options:

--size WIDTHxHEIGHT 
  Set the video size, e.g. "1280x720". Default is the device's main

  display resolution (if supported), 1280x720 if not. For best results,

  use a size supported by the AVC encoder.
--bit-rate RATE 
  Set the video bit rate, in bits per second. Value may be specified as

  bits or megabits, e.g. '4000000' is equivalent to '4M'. Default 20Mbps.
--bugreport 
  Add additional information, such as a timestamp overlay, that is helpful

  in videos captured to illustrate bugs.
--time-limit TIME 
  Set the maximum recording time, in seconds. Default / maximum is 180.
--verbose Display interesting information on stdout. Wed, 07 Apr 2021 03:39:46 -0400
